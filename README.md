## 🌟 核心特性

| 特性 | 说明 |
|------|------|
| 🔍 **像素级识别** | 通过屏幕色块实时解析游戏状态，精度达 1/255 |
| ⚡ **高性能解析** | 在 Ryzen 9700X 上可达 100FPS 解析速度 |

| 🔧 **模块化架构** | 插件-解析器-执行器三层分离设计 |
